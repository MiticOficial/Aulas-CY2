LISTAS

Uma forma de exibir várias informações na tela é por meio de listas. A seguir são apresentados dois tipos de listas que você pode utilizar nas suas páginas:

Listas Ordenadas(ol):

Nesse tipo de lista, os itens possuem uma ordem, pode ser a prioridade de cada um, o preço, a data que foram inseridos na página. Cabe a você a organização deles, utilizando os atributos: start – Define a partir de qual número a listagem começa type – Tipos de caracteres ordenados: A, a, I, i, 1:

Exemplo:

  1. teste
  2. teste
  3. teste
  1. teste
  2. teste
  3. teste
  1. teste
  2. teste
  3. teste
